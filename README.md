# Device Insurance Attach % Analysis & January Forecast

## Project Overview

This project analyzes device insurance attach percentage trends across 163 retail stores for Jumbo & Company from August to December, identifies store-level performance patterns, and forecasts January attach % using a weighted trend-based approach. The analysis focuses on uncovering execution gaps, high-performing stores, and actionable business insights to improve insurance adoption.

## Key Objectives

- Analyze month-wise and store-level attach % performance

- Categorize stores based on performance (Champion, Stable, Opportunity, Critical)

- Identify volatility and execution-driven differences

- Forecast January attach % using recent trends

- Provide data-backed recommendations for business improvement


## Tools & Technologies Used

### Microsoft Excel

- Data cleaning & preparation

- Pivot tables & calculated metrics

- Trend and volatility analysis

### Data Visualization

- Line charts, bar charts, pie charts for insights

- Business Analytics Techniques

- Store segmentation

- Weighted trend-based forecasting

- Performance benchmarking

## Key Insights

- Overall attach % improved steadily from 13% (Aug) to 22% (Dec)

- A small set of stores drives a disproportionate share of insurance adoption

- Performance varies more by store execution than geography

- Majority of stores fall under Opportunity & Critical categories, indicating quick-win potential

## Outcome

The analysis enables targeted store-level interventions, better incentive alignment, and realistic attach % targets to drive insurance adoption sustainably.
